Navigate:
=========

Goto Anything (File/Symobol)    Ctrl + P                      
Upper case:                     Ctrl + K, Ctrl + U            
lower case:                     Ctrl + K, Ctrl + L            
Select Word under Cursor        Ctrl + D                      
Go to Line                      Ctrl + G                      
Follow Symbol under Cursor      F12  
Go Back                         Alt + Left
Go Forward                      Alt + Right


Select:
=======
Select the hole line            Ctrl + L
Expand selection to scope       Ctrl + Shift + Space


Editing:
========

Duplicate Line (Copy Line Down) Ctrl + Shift + D
QuickFix                        Alt + Return
Toggle Comment                  Ctrl + Shift + 7
Delete Line                     Ctrl + Shift + K
